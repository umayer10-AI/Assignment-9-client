# 🩺 DocAppoint

DocAppoint is a modern healthcare appointment booking platform where users can search doctors, book appointments, manage bookings, and update their profiles easily.

## 🌐 Live Site

🔗 Live Website: https://assignment-9-client-three.vercel.app

---

## 🚀 Features

- 🔍 Search doctors by specialty or name
- 📅 Book appointments with preferred date & time
- 👨‍⚕️ View detailed doctor profiles and availability
- ✏️ Update and manage booked appointments
- 🔐 Authentication system with Email/Password and Google Login
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🎨 Modern UI built with Tailwind CSS and HeroUI
- 👤 User dashboard with profile management
- ⚡ Fast and optimized using Next.js App Router
- 🛡️ Protected private routes using middleware authentication

---

## 🛠️ Technologies Used

- Next.js
- React.js
- Tailwind CSS
- HeroUI
- MongoDB
- Better Auth
- React Hook Form
- Swiper.js
- React Hot Toast
